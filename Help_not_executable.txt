# entwurf Template
######################
# grundsätzliche Befehle
######################
# SET - setzt wert
# ASK - fragt wert mit textfeld ab 
# OPT - fragt wert anhand vorgegebener werte <wert1,wert2,wert3{ <optional> map1,map2,map3}>
# SELECT - fragt wert anhand selectllista aller verfügbaren devices/werte ab
# EVENT - fragt wert anhand des eventmonitors ab
# INQ - leitet optionale zweige ein
# VARASK - Setzen einer Variable ( Variable wird abgefragt )
# VARSET - direktes setzen einer Variable
# VAREVENT - setzen einer Variable aus dem Eventmonitor
# VARDEVICE - setzen einer Variable aus der Deviceauswahl
# ATTR - Attribut setzen

######################
# abzufragende Felder
######################
# format : <opt> >> <feld> >> <angezeigter text>
######################

# INFO - Infotext
# Trigger_device  - auslösendes Gerät
# Time_on  - zeit schaltet mswitch an
# Time_off - zeit schaltet mswitch aus
# Time_cmd1 - zeit ausführung cmd1
# Time_cmd2 - zeit ausführung cmd1
# Trigger_condition - Bedingung für Trigger ( nicht events )
# Device_to_switch - zu schaltendes Gerät

# Mswitch_on - fragt Event ab ( MSwitch anschalten und Kanal 1 ausführen )
# Mswitch_off - fragt Event ab ( MSwitch ausschalten und Kanal 2 ausführen )
# Mswitch_cmd1 - fragt Event ab ( Kanal 1 ausführen )
# Mswitch_cmd2 - fragt Event ab ( Kanal 2 ausführen )

# comand_cmd1 - Inhalt Kommandozweig 1
# comand_cmd2 - Inhalt Kommandozweig 2
# delay_cmd1 - verzögerte Befehlsausführung kanal 1
# delay_cmd2  - verzögerte Befehlsausführung kanal 2
# condition_cmd1 - schaltbedingung Kanal 1
# condition_cmd2 - schaltbedingung Kanal 2
# cmd_repeat - wiederholung der Kanälq 1 und 2
# cmd_repeat_time - zeit in sekunden zwischen wiederholungen

######################

# INQ - Abfrtage Option
# BEISPIEL: INQ>>TRIGGER>>TIME>>Eventsteuerung,Zeitsteuerung>>Soll das MSwitch Eventgesteuert oder Zeitgesteuert sein 

# ATTR - Attribut (SET,OPT,ASK,SELECT)
# Beispiel: ATTR>>SET>>disable>>1

# VAR
# Beispiel: VAREVENT>>$TEST>>Text

# REPEAT - Codewiederholung / Anzahl der zu wiederholenden Templatezeilen / ( Abfrage ja/nein ) 
# BEISPIEL: REPEAT>>3>>Soll ein weiteres Gerät geschaltet werden ? ( ja wiederholt die letzten 3 codezeilen inc. REPEAT )
